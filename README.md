# bamazon

An Amazon-like storefront using MySQL.

**bamazonCustomer**
Take in orders from customers and deplete stock from the store's inventory.

**bamazonManager**
Manager is able to 
- view products for sale 
- View vow inventory 
- Add to inventory
- Add new product
